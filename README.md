# Sample-BI-Project
Financial spend analysis project using SQL and Tableau, built on simulated fintech data.

### Business Questions Answered:
1. Which companies spend the most?
2. How does spending change month over month?
3. Which companies exceed their budget?
4. Which transactions were declined?
5. Which employee has the most expense reports?
6. Which employees have above average expense amounts?

### Tools Used:
- SQL (SQLite / DB Browser)
- Tableau Public
- VS Code
